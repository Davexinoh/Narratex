# Narratex System Architecture

Narratex is designed as a modular narrative intelligence pipeline.

---

## Pipeline

Market Signals
↓
Signal Collector
↓
Narrative Extractor
↓
Momentum Engine
↓
Prediction API
↓
Dashboard

---

## Components

### Signal Collector
Collects market data from crypto exchanges.

### Narrative Extractor
Clusters signals into emerging narratives.

### Momentum Engine
Calculates narrative confidence and growth.

### API Server
Serves predictions through REST endpoints.

### Dashboard
Displays narrative intelligence visually.
